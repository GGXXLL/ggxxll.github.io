Mysql order by 和 limit 语句导致慢 sql

Sep 19, 2022
阅读时长: 2 分钟

问题

order by 和 limit 造成优化器选择索引错误

原因

MySQL 优化器认为在 limit 值较小的情况下，走主键索引能够更快的找到那一条数据，并且如果走联合索引需要扫描索引后进行排序，而主键索引天生有序，所以优化器综合考虑，走了主键索引。
